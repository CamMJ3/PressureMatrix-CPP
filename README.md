# Resolución de una Matriz A en C++

Implementación de un matriz A y un vector B con funciones de visualización de contenido, así como su resolución mediante los métodos de Gauss y de Gauss-Jordan con la inversa de la matriz. Tanto la matriz como el vector representan la presión en un tanque de almacenamiento.

## Características

Visualización de... 
- La matriz A y el vector B.
- La resolución del sistema de ecuaciones lineales mediante el método de Gauss.
- La matriz inversa de A y la resolución del sistema mediante el método de Gauss-Jordan.
