Detailed Description

Define los constructores para resolver sistemas de ecuaciones lineales mediante matrices.

El presente archivo contiene los prototipos de las funciones necesarias para imprimir el contenido de la matriz A y el vector B del problema situado, así como para resolver la matriz A utilizando el método de eliminación de Gauss, obtener la matriz inversa de A, y multiplicar la matriz A por el vector B, con el próposito de encontrar el valor de las incógnitas del sistema de ecuaciones lineales.

vector< double > matrizInversa ( vector< vector< double > > A,
const vector< double > & b )

Obtiene la matriz inversa de A utilizando el método de eliminación de Gauss-Jordan.

Se utiliza la matriz identidad para transformar la matriz A en una matriz aumentada, posteriormente, se normaliza cada fila (el pivote se vuelve 1) y elimina los coeficientes debajo del pivote (los coeficientes debajo del pivote se vuelven 0), a tal modo que la matriz A se transforme en una matriz identidad.

La matriz identidad, una vez transformada, se convierte en la matriz inversa de A, la cual es necesaria para encontrar las incógnitas del sistema de ecuaciones lineales al multiplicarla por el vector B.

vector< double > metodoGauss ( vector< vector< double > > A,
vector< double > b )

Resuelve la matriz A mediante el método de eliminación de Gauss, devolviendo el valor de las incógnitas del sistema.

Transforma la matriz A, primero, en una matriz aumentada de A que incluye el vector B, posteriormente, se transforma en una matriz triangular superior, normalizando cada fila (el pivote se vuelve 1) y eliminando los coeficientes debajo del pivote (los coeficientes debajo del pivote se vuelven 0).

Para encontrar el valor de las incógnitas, se sustituye a partir de la última fila hacia arriba.

Multiplica la matriz A por el vector B, lo cual es útil para calcular el producto de la matriz inversa de A por el vector B.

Se multiplica cada fila de la matriz A por el vector B, sumando los productos para encontrar el valor de cada incógnita del sistema de ecuaciones lineales.
