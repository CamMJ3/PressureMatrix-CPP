\doxysubsection{Detailed Description}
El programa completo para la manipulación de la matriz A y el vector B. 

Las opciones son las siguientes:\+
\begin{DoxyItemize}
\item Opción:\+ 1 Visualizar la matriz A y el vector B.
\item Opción 2:\+ Visualizar la resolución del sistema de ecuaciones lineales mediante el método de Gauss.
\item Opción 3:\+ Visualizar la matriz inversa de A y la resolución del sistema mediante el método de Gauss-\/\+Jordan.
\item Opción 4:\+ Salir del programa. 
\end{DoxyItemize}
